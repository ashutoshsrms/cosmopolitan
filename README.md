# Cosmopolitan A socila network App


## Tech Stack

- Node.js
- Express.js
- MongoDB
- React

## Setup

1. Clone the repository.
2. Install dependencies:

    ```bash
    cd backend
    npm install

    cd frontend
    npm install

    ```

3. Start the server:

    ```bash
    cd backend
    npm run dev
        ```

4. Start the frontend:

    ```bash
    cd frontend
    npm start
        ```
